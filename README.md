
## osaas 

> 多功能，易扩展的SAAS全栈脚手架。   

* center -  业务管理模块  
* gateway - 网关模块  
* identity - 用户租户管理，RBAC权限  
* oauth - 认证中心
* integrate - 三方能力整合模块 
* common - 公共工具包  


| 依赖                   | 版本         |
| ---------------------- |------------|
| Spring Boot            | 2.7.5       |
| Spring Authorization Server | 0.3.1  |
| Spring Cloud Gateway        | 3.1.4  |
| OpenApi3.0 SpringDoc        | 1.6.11 |
| JPA                         | 2.7.5  |
